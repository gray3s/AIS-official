This project is intended to give hard feedback from an agentic AI agent. 
What I have provided here is just a skeleton example. 
The point is to make the agent "hallucinate"...
and then have it explain why it did so and what it would do to prevent it from further "hallucionations".
Modify the rules and scope to validate the agent configuration as you see fit.

You may want to have your agent write the responses to the given prompts 
to a single text-file for easy download and review.

Note that the content of the repository files was created entirely by chatgpt. 
I just gave it the seed rules, scope and prompts. 
The rest of it came from ChatGPT itself.

Enjoy
